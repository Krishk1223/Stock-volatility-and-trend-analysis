# Stock Analysis

Stock Analysis explores stock market data, analyses a portfolio of stocks and runs GARCH and ARIMA models to 
help simulate future paths the stock may take. The portfolio is then optimised through the framework of Modern
Portfolio Theory by maximising the Sharpe Ratio.

## Features

- Load historical price data
- Compare multiple stocks
- Calculate basic indicators and summary metrics
- Visualise trends over time
- Optimise Portfolio

## Getting Started

### Prerequisites

- Python 3.11 or newer
- `pip`

### Installation

Copy repository url from above then:

```bash
git clone <repository-url>
cd stock_analysis
pip install -r requirements.txt
```

## Usage

Run the project entry point if available:

```bash
python main.py
```

## Example Workflow

1. Choose one or more tickers
2. Fetch historical data
3. Analyze price movement and indicators
4. Review charts and comparison results
5. Compare stock portfolio expected return and sharpe ratio to view if optimised portfolio is useful.
